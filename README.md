# 🌱 EcoEnergy Dashboard

A modern Streamlit-based carbon emission tracking and energy management platform with gamification features.

## ✨ Features

- **Real-time Energy Monitoring** - Track electricity, gas, and water consumption
- **Carbon Footprint Analysis** - Calculate and visualize CO2 emissions
- **ML-Powered Predictions** - Forecast future energy usage patterns
- **Gamification System** - Earn badges and compete on leaderboards
- **Smart Recommendations** - AI-driven energy saving suggestions
- **Dark Theme UI** - Modern glass morphism design

## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/Dheemanth49/carbon-emission-.git
cd carbon-emission-

# Install dependencies
pip install -r requirements.txt

# Run application
streamlit run streamlit_app.py
```

Access at `http://localhost:8501`

## 🛠️ Tech Stack

- **Streamlit** - Web framework
- **SQLite** - Database
- **Scikit-learn** - Machine learning
- **Plotly** - Interactive charts
- **Pandas/NumPy** - Data processing
